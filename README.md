# 🚁 低空经济无人机路径规划系统

基于A*搜索算法的智能路径规划系统，为无人机在复杂城市环境中寻找安全高效的最优路径。

## ✨ 项目特色

- 🚀 **毫秒级路径规划** - 5ms内完成复杂环境路径计算
- 🏙️ **多约束环境建模** - 支持建筑物障碍物和禁飞区
- 📊 **实时可视化** - 完整展示算法搜索过程和最优路径
- 🎯 **低空经济应用** - 面向无人机物流和城市空中交通场景

## 🛠️ 技术栈

- **编程语言**: Python 3.8+
- **核心算法**: A* (A-Star) 搜索算法
- **空间计算**: NumPy, 自定义网格环境
- **可视化**: Matplotlib
- **应用领域**: 低空经济、无人机路径规划
